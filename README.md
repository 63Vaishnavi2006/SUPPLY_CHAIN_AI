# SUPPLY_CHAIN_AI

# 🚀 AI-Based Supply Chain Optimization System

## 📌 Project Overview
The **AI-Based Supply Chain Optimization System** is a simple web-based application developed using **Python and Flask**.  
It helps in determining whether the available stock is sufficient or low by comparing **predicted demand** and **current stock** values.

This project is designed as a **mini project / internship project** to demonstrate the practical use of Artificial Intelligence concepts with a user-friendly interface.

---

## 🎯 Problem Statement
Many organizations face issues such as:
- ❌ Overstocking
- ❌ Understocking
- ❌ Manual inventory checking
- ❌ Poor decision-making

There is a need for a simple system that helps in **quick inventory status checking**.

---

## ✅ Solution
This project provides:
- 🧠 Intelligent comparison of demand and stock
- 🌐 Web-based user interface
- ⚡ Instant result display
- 👩‍💻 Easy-to-use design for beginners
